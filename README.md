# 한국 주식 바닥 스크리너 (Korea Bottom Screener)

코스피·코스닥 전종목을 매일 자동으로 훑어 **바닥 신호**가 강한 종목을 뽑아주는 도구입니다.
서버를 상시 켜둘 필요 없이 GitHub Actions가 매일 한 번 계산하고, 결과를 웹 화면으로 보여줍니다.

## 구성

```
signals.py                 # 점수 엔진 (7개 신호 → 0~100점). 네트워크 불필요, 단독 테스트 가능
screener.py                # pykrx로 데이터 수집 → 채점 → results.json 저장
requirements.txt
.github/workflows/daily.yml # 매일(평일 KST 20:00) 자동 실행
docs/index.html             # 결과 화면 (results.json을 읽어 순위 표시)
results.json               # 매 실행마다 갱신되는 결과 (Actions가 커밋)
```

## 7개 신호 (동일 가중, 각 0~100점)

1. **거래량 고갈** — 최근 20일 거래량 중앙값 ÷ 과거 120일 중앙값 (낮을수록 고점)
2. **기관·외국인 매집** — 20일 누적 순매수 ÷ 유통시총, 주가가 조용할수록 가중
3. **공매도 감소** — 현재 공매도잔고비중 ÷ 3개월 최고 (낮을수록 고점)
4. **PBR 역사적 저점** — 5년 PBR 밴드 내 백분위 (낮을수록 고점)
5. **배당수익률(조건부)** — 배당주 한정, 배당성향 80%↑·삭감이력은 함정으로 배제
6. **상대강도** — 60일 수익률의 코스피 대비 초과분 (지수보다 버틸수록 고점)
7. **변동성 수축(밴드 스퀴즈)** — 20일 볼린저밴드폭이 자체 과거 히스토리 대비 얼마나 좁은지
   (좁을수록=가격이 눌려 다져질수록 고점). 기존 OHLCV만으로 계산, 새 데이터소스 불필요.

> 신용잔고·내부자/자사주 매수는 KRX 공식/비공식 API, 네이버금융 어디에도 무료로 종목별
> 조회할 방법을 찾지 못해 빠져 있습니다. (재무건전성은 DART Open API 연동으로 가능할 수 있음)

## 설치·배포 (처음 하는 사람 기준)

1. **이 폴더를 새 GitHub 저장소에 올린다.** (Claude Code로 하면 자동)
2. 저장소 **Settings → Actions → General**에서 workflow 쓰기 권한 허용.
3. **KRX 로그인 계정 준비 (2025-12-27부터 필수, 무료)**: [data.krx.co.kr](https://data.krx.co.kr)에서
   네이버/카카오로 간편가입 후, 저장소 **Settings → Secrets and variables → Actions**에서
   `KRX_ID`, `KRX_PW` 시크릿을 등록. (없으면 모든 조회가 실패합니다)
4. **Actions 탭 → daily-screener → Run workflow**로 첫 실행. (몇 분 소요)
   - 첫 실행에서 휴장일·결측 등으로 조정이 필요할 수 있습니다(정상).
5. 실행이 끝나면 `results.json`이 생기고 `docs/results.json`으로 복사됩니다.
6. **화면 띄우기**: 저장소 **Settings → Pages**에서 Source를 `main` 브랜치 `/docs` 폴더로 설정.
   `https://<계정>.github.io/<저장소>/` 주소가 생기면 폰 브라우저에서 열고
   "홈 화면에 추가"로 아이콘처럼 쓸 수 있습니다.

## 로컬에서 점수 로직만 확인

```bash
python signals.py      # 가짜 데이터로 7개 신호·종합점수 검증 (네트워크 불필요)
```

## 다음 단계

- **백테스트**: 과거 바닥 사례(예: 2022년 반도체 저점)에 임계값을 대입해 신호가 실제로 켜졌는지 역검증하고 임계값·가중치를 조정.
- **2차 신호**: 내부자/자사주·재무건전성(부채비율·이자보상배율, DART Open API 필요) 추가.
- **파라미터**: `screener.py` 상단 상수(임계값·상위 N)로 조정.

> ⚠️ 이 도구는 후보를 **발굴**하는 스크리너이며 매수·매도 신호가 아닙니다. 최종 판단은 사용자 본인에게 있습니다.
> KRX 공개데이터 가공물을 다수에게 공개할 경우 데이터 이용약관을 확인하세요.
